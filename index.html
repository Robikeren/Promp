Bagian 4 — Fungsi adalahPrima (Fungsi + If-Else + For Loop)

Sekarang kita masuk ke bagian inti program — fungsi untuk mengecek apakah sebuah bilangan adalah prima. Fungsi ini berdiri sendiri di luar kelas, sehingga bisa dipanggil dari mana saja. Di sinilah tiga topik sekaligus diimplementasikan: Fungsi, If-Else, dan For Loop.

Kita mulai dengan menulis deklarasi fungsinya. Fungsi ini menerima satu parameter integer n, dan mengembalikan nilai boolean — true jika prima, false jika bukan.

⌨ ketik ini
bool adalahPrima(int n) {

Percabangan if pertama: bilangan yang kurang dari atau sama dengan 1 pasti bukan prima. Kita langsung kembalikan false.

⌨ ketik ini
  if (n <= 1) return false;

Percabangan kedua: angka 2 adalah satu-satunya bilangan prima yang genap. Kita tangani sebagai kasus khusus dan langsung kembalikan true.

⌨ ketik ini
  if (n == 2) return true;

Percabangan ketiga: semua bilangan genap selain 2 pasti bukan prima karena habis dibagi 2. Kita kembalikan false untuk menghemat komputasi.

⌨ ketik ini
  if (n % 2 == 0) return false;

Sekarang For Loop. Kita cek apakah n punya pembagi selain 1 dan dirinya sendiri. Kita mulai dari 3 dan loncat 2 angka — hanya bilangan ganjil. Kondisi berhenti i kali i lebih dari n, setara dengan i lebih dari akar kuadrat n. Ini adalah optimasi agar tidak perlu memeriksa semua angka hingga n.

⌨ ketik ini — selesaikan fungsi
  for (int i = 3; i * i <= n; i += 2) {
    if (n % i == 0) return false;
  }
  return true;
}

Jika loop selesai tanpa menemukan pembagi, berarti bilangan tersebut prima dan kita kembalikan true.

Bagian 5 — Fungsi primaDalamRentang (Fungsi + For Loop)

Fungsi kedua adalah primaDalamRentang. Fungsi ini menerima dua parameter — batas awal dan batas akhir — kemudian mengumpulkan semua bilangan prima dalam rentang tersebut ke dalam sebuah List dan mengembalikannya.

⌨ ketik ini
List<int> primaDalamRentang(int awal, int akhir) {
  List<int> hasil = [];
  for (int i = awal; i <= akhir; i++) {
    if (adalahPrima(i)) hasil.add(i);
  }
  return hasil;
}

Kita buat list kosong bernama hasil. Lalu dengan For Loop, kita iterasi setiap angka dari awal hingga akhir. Setiap angka kita masukkan ke fungsi adalahPrima. Kalau hasilnya true, angka tersebut kita tambahkan ke list. Di akhir, list dikembalikan.

Bagian 7 — TabCekAngka (If-Else + Fungsi)

Sekarang kita buat Tab pertama — TabCekAngka. Tab ini adalah StatefulWidget karena tampilannya berubah sesuai input dan hasil pengecekan. Di sinilah topik If-Else dan Fungsi bekerja bersama.

Kita tulis dulu kerangka StatefulWidget-nya beserta variabel-variabel state yang dibutuhkan.

⌨ ketik ini — deklarasi kelas

class _TabCekAngkaState extends State<TabCekAngka> {

TextEditingController digunakan untuk mengambil teks dari input field. _hasilTeks menyimpan pesan yang ditampilkan ke pengguna. _hasilPrima menyimpan nilai boolean hasil pengecekan — null berarti belum dicek.

Sekarang kita tulis metode _cek. Metode ini dipanggil saat pengguna menekan tombol. Di sini If-Else berperan penting untuk validasi input dan menampilkan hasil yang berbeda.

⌨ ketik ini — metode _cek()
  void _cek() {
    final input = int.tryParse(_ctrl.text.trim());
    if (input == null) {
      setState(() {
        _hasilTeks = 'Input tidak valid! Masukkan angka bulat.';
        _hasilPrima = null;
      });
      return;
    }
    final prima = adalahPrima(input);
    setState(() {
      _hasilPrima = prima;
      if (prima) {
        _hasilTeks = '$input adalah BILANGAN PRIMA';
      } else {
        _hasilTeks = '$input BUKAN bilangan prima';
      }
    });
  }

int.tryParse mencoba mengubah teks menjadi integer. Kalau gagal hasilnya null. Percabangan if pertama menangani input tidak valid. Kalau valid, kita panggil fungsi adalahPrima. setState memastikan Flutter memperbarui tampilan setelah nilai berubah. Percabangan if-else berikutnya menentukan teks yang ditampilkan sesuai hasil.

Bagian 8 — TabRentang (While Loop + If-Else + Fungsi)

Tab kedua adalah TabRentang. Di sinilah topik While Loop diimplementasikan sebagai cara alternatif melakukan perulangan. Pengguna memasukkan batas awal dan batas akhir, lalu program menampilkan semua bilangan prima dalam rentang tersebut.

Kita tulis dulu kerangka kelas dan variabel state-nya.

⌨ ketik ini — deklarasi TabRentang
class TabRentang extends StatefulWidget {
  const TabRentang({super.key});
  @override
  State<TabRentang> createState() => _TabRentangState();
}

class _TabRentangState extends State<TabRentang> {
  final TextEditingController _awalCtrl = TextEditingController(text: '1');
  final TextEditingController _akhirCtrl = TextEditingController(text: '50');
  List<int> _hasil = [];
  String? _pesan;
  bool _sudahCari = false;

Kita punya dua controller untuk dua input field — batas awal dan batas akhir, dengan nilai awal 1 dan 50. _hasil adalah list yang menyimpan bilangan prima yang ditemukan. _sudahCari menentukan kapan hasil mulai ditampilkan.

Sekarang metode _cari dengan While Loop. Berbeda dengan For Loop yang increment-nya otomatis, While Loop mengharuskan kita mengelola variabel counter secara manual.

⌨ ketik ini — metode _cari() dengan While Loop
  void _cari() {
    final awal = int.tryParse(_awalCtrl.text.trim());
    final akhir = int.tryParse(_akhirCtrl.text.trim());
    if (awal == null || akhir == null) {
      setState(() {
        _pesan = 'Input tidak valid!';
        _hasil = [];
        _sudahCari = true;
      });
      return;
    }
    if (awal > akhir) {
      setState(() {
        _pesan = 'Batas awal tidak boleh lebih besar dari batas akhir.';
        _hasil = [];
        _sudahCari = true;
      });
      return;
    }
    List<int> temp = [];
    int i = awal;
    while (i <= akhir) {
      if (adalahPrima(i)) temp.add(i);
      i++;
    }
    setState(() {
      _hasil = temp;
      _pesan = null;
      _sudahCari = true;
    });
  }

Kita validasi dua input dulu dengan If-Else. Kalau salah satu null berarti bukan angka, kita tampilkan pesan error. Kalau awal lebih besar dari akhir juga tidak valid. Lalu kita masuk ke While Loop. Variabel i dimulai dari awal, loop berjalan selama i masih kurang dari atau sama dengan akhir. Di dalam loop kita panggil fungsi adalahPrima. Kalau true, angka ditambahkan ke list temp. Setelah itu i kita tambah 1 secara manual. Inilah perbedaan utama While Loop dengan For Loop.
